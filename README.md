# Pharmacy Industry Trend Analysis Dashboard

## Project Overview

This project analyzes pharmacy industry trends between 2021 and 2026 using SQL and Power BI. The dashboard provides insights into pharmacy openings, closures, distance seller activity, and net industry growth.

## Tools Used

* SQL
* Power BI
* Microsoft Excel

## Key Features

* Year-wise pharmacy trend analysis
* Pharmacy openings and closures tracking
* Distance seller trend analysis
* Pharmacy size-wise closure analysis
* Net change calculations
* Interactive dashboard with slicers

## Dashboard KPIs

* Total Pharmacies
* Physical Pharmacies
* Distance Sellers
* Total Net Change

## Key Insights

* Large pharmacies recorded the highest closures.
* Pharmacy closures peaked in 2023.
* Net pharmacy growth was negative in multiple years.
* Distance seller activity fluctuated significantly across the study period.
